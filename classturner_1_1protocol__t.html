Detailed Description

template<typename ProtocolTraits>
class turner::protocol_t< ProtocolTraits >

Generalised protocol description class.

Concrete protocols define own ProtocolTraits and wrap it into protocol_t which provides common API for message parsing.

Return pointer to message instance wrapping raw network format in range [first, last).

Returned pointed object is not actually instantiated but overlayed onto specified memory range [first, last). Each time any of any_message_t method is invocated, it parses fields relative to this.

If during call to parse() message validation fails, nullptr is returned and error is set to code describing failure.
