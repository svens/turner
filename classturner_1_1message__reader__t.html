Return true if message's claimed integrity is same as calculated with integrity_calculator.

On failure, set error code identifying problem and return false.

Possible errors:

  • turner::errc::attribute_not_found: message does not have MESSAGE-INTEGRITY attribute
  • turner::errc::unexpected_attribute_length: invalid integrity length
  • turner::errc::unexpected_attribute_value: claimed integrity is not valid
