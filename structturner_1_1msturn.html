Detailed Description

MS-TURN protocol requests/attributes.

See also
https://docs.microsoft.com/en-us/openspecs/office_protocols/ms-turn
Note
MS-TURN requires Magic Cookie to be 1st attribute immediately after message header. To simplify reader/writer, we treat it as binary blob instead i.e. Magic Cookie contains attribute Type and Length fields.
