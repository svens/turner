Protocol type common to all attribute types.

It is compile-time error to combine attribute types from different protocol into single type list unless protocols are convertible (like STUN <-> TURN)

Returns unary lambda that checks if specified uint16_t attribute type has value in comprehension required range and is not listed in attribute_type_list::comprehension_required()

Differently put, passing this lambda as unread argument for message_reader::not_read() collects attribute types in comprehension required range that are not read when using this argument_type_list
